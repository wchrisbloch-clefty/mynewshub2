## Track 1 — Covered Calls / Wheel Strategy (Activate Immediately)

**What**: Sell covered calls on existing stock positions to generate weekly or monthly premium income. If assigned (stock called away), sell cash-secured puts at a lower strike to re-enter the position. The wheel continues indefinitely.

**Why now**: If CB holds 100+ shares of any position, this generates income from assets CB already owns — no new capital required. One trade. Set it. Collect the premium.

**Income model**: 1–3% monthly premium on position value depending on volatility and strike selection.
- $10,000 position in a moderately volatile stock → $100–$300/month
- $50,000 deployed across positions → $500–$1,500/month in premium income

**Autonomy level**: One trade per weekly or monthly cycle. Minimal screen time. Not day trading.

## Track 3 — Tax-Advantaged Account Optimization

**What**: Maximize tax-advantaged contributions before deploying in taxable accounts. Every dollar saved in taxes is a dollar that compounds — the highest guaranteed return in CB's portfolio.

**Why this comes before deployment decisions**:
High-yield income-producing assets (BDCs, MLPs, high-dividend equities) generate taxable events every quarter. Holding them inside tax-advantaged accounts eliminates that drag.

**Account Optimization Checklist**:
- [ ] Roth IRA — maxing contributions? ($7,000/year 2025 if under 50; $8,000 if 50+)
- [ ] Solo 401(k) — if self-employment income exists, this allows the highest contribution limit available to an individual
- [ ] SEP-IRA — if sole proprietor, simpler alternative to Solo 401(k)
- [ ] HSA — if eligible: triple tax advantage (pre-tax contribution, tax-free growth, tax-free qualified withdrawal)
- [ ] Taxable brokerage — deploy here only after tax-advantaged accounts are maxed

**Newsletter content note**: Keeping up with tax advantaged changes → this directly feeds PI_AI_DIGITAL newsletter content AND informs CB's own positioning. CB's knowledge of these changes is a product and a personal financial advantage simultaneously.
