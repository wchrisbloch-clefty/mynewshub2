# Passive Income — Master Pipeline

## The Rule
Every idea enters here first. Nothing routes to a platform document without passing through intake.

## Pipeline Stages
```
INTAKE → FILTER → ROUTE → ACTIVE → COMPLETE / SHELVED
```

## Autonomous Filter (Applied at Intake)
Four questions. Answer all four before routing:
1. Does this generate revenue without CB's active time after setup?
2. What is the realistic time to first dollar?
3. What is CB's existing asset alignment? (High / Medium / Low)
4. What is the build requirement? (None / Light / Heavy)
