# One-Page Cheat Sheet Generator

Executive leadership style. Scannable in 5 minutes. No walls of text.

## Step 1 — Establish Scope

Determine:
- **Topic**: [What is being summarized]
- **Audience lens**: [CB using this for himself / presenting to others / prepping for a meeting]
- **Depth**: [Surface familiarity / working knowledge / expert reference]
- **Format priority**: [Maximum bullets / include diagram / include examples / all three]

## Step 2 — Extract Core Content

Identify:
1. The 3–5 concepts that define the topic
2. The key vocabulary (5–8 terms max)
3. The most useful framework or mental model
4. 2–3 real-world examples that make it concrete
5. The 1–2 things most people get wrong

## Step 3 — Build the Cheat Sheet

Structure:

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[TOPIC] — QUICK REFERENCE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

CORE IDEA (one sentence)
[The single most important thing to understand]

KEY CONCEPTS
• [Concept 1]: [Definition in plain English]
• [Concept 2]: [Definition in plain English]
• [Concept 3]: [Definition in plain English]
• [Concept 4]: [Definition in plain English]
• [Concept 5]: [Definition in plain English]

THE FRAMEWORK
[Visual or structured representation — table, flow, or diagram]

HOW IT WORKS IN PRACTICE
• [Real example 1 — specific, not generic]
• [Real example 2 — tied to CB's world where possible]

COMMON MISTAKES
✗ [What people get wrong #1]
✗ [What people get wrong #2]

CB EDGE
→ [How this connects to CB's professional or financial context]
→ [The asymmetric insight most people miss]

KEY NUMBERS / METRICS (if applicable)
[The 3–5 numbers worth memorizing]

ONE QUESTION TO ASK
"[The question that reveals whether someone actually understands this topic]"
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## Step 4 — Quality Check

Before delivering:
- ☐ Readable in under 5 minutes
- ☐ No sentence longer than 15 words
- ☐ Every concept has a plain-English definition
- ☐ At least one example tied to CB's actual context
- ☐ Fits on one page (600 words maximum)
- ☐ The "CB Edge" section is specific, not generic

## Hard Rules
- Never write a cheat sheet that reads like a textbook
- If the topic can't be summarized on one page, flag the sub-topic that should be the actual focus
- Always include the "CB Edge" — a cheat sheet without a specific angle for CB's situation is just Wikipedia
