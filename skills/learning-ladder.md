# Learning Ladder

Five levels. Clear milestones. You know exactly where you stand.

## Step 1 — Define the Topic and End Goal

Establish:
- **Topic**: [What is being laddered]
- **CB's current level**: [Zero / Level 1–2 / Level 3+]
- **Target level**: [What does CB need to reach — and why]
- **Time available**: [Connects to 20-hour plan if applicable]

## Step 2 — Build the Five Levels

| Level | Name | What You Know | What You Can Do | Milestone Test |
|---|---|---|---|---|
| 1 | **Foundation** | Core vocabulary, basic concepts | Explain it to a non-expert | Can you define the 5 key terms without looking them up? |
| 2 | **Functional** | How it works, common patterns | Apply it to a real scenario | Can you solve a basic problem from scratch? |
| 3 | **Practitioner** | Edge cases, tradeoffs, context | Make decisions with it | Can you evaluate two options and explain why one is better? |
| 4 | **Advanced** | Deep mechanics, second-order effects | Spot what others miss | Can you identify when the standard approach fails? |
| 5 | **Expert** | Mental models, systems thinking | Teach it, challenge it, extend it | Can you teach someone else to Level 3 in one session? |

## Step 3 — Level-Specific Content

For each level, specify:

**Level 1 — Foundation**
- Key concepts to internalize: [list]
- Best first resource: [specific]
- Time investment: [hours]
- Common trap at this level: [what beginners get wrong]

**Level 2 — Functional**
- Skills to develop: [list]
- Best practice resource: [specific]
- Time investment: [hours]
- Common trap: [what Level 1 people miss when moving up]

**Level 3 — Practitioner**
- Frameworks to master: [list]
- Best real-world resource: [case studies, practitioners]
- Time investment: [hours]
- Common trap: [what functional people think they know but don't]

**Level 4 — Advanced**
- Mental models to build: [list]
- Best advanced resource: [specific]
- Time investment: [hours]
- Common trap: [the overconfidence zone]

**Level 5 — Expert**
- Synthesis points: [how everything connects]
- Community / peer group: [where experts gather]
- Time investment: [ongoing]
- Marker: [the question only an expert would think to ask]

## Step 4 — CB Context Overlay

For the topic, flag:
- Which level is sufficient for CB's stated purpose
- Where CB's existing knowledge creates a shortcut (energy industry edge, financial framework, BD experience)
- Which level unlocks a specific financial or professional opportunity

## Step 5 — Output Format

---
**LEARNING LADDER: [TOPIC]**
*CB's current level: [X] | Target level: [Y] | Purpose: [Z]*

[5-level table]

**Your Shortcut:** [Where CB's existing knowledge compresses the ladder]

**Target Level for Purpose:** Level [X] — here's what that unlocks: [specific outcome]

**Next Action:** [The single first step to move from current level to next]

---

## Hard Rules
- Never make Level 1 too hard — the ladder fails if the first rung is out of reach
- Always tie the target level to CB's specific purpose — "reach Level 5" is almost never the right answer
- Flag where CB's energy/finance/BD background creates genuine level-skipping opportunities
